一、RAM&ROM的区别

  1. 最大的区别是RAM断电以后不会保存数据；
  2. 无论是电脑还是手机，容量最小的一定是RAM，容量最大的一定是储存(闪存)ROM，前者是运行内存，后者可以理解为硬盘(闪存)。

返回目录

二、RAM特点

  1. RAM 又称作"随机存储器";
  2. 与CPU直接交换数据的内部存储器，也叫主存；
  3. 它随时可读可写，而且速度很快，通常作为运行程序的临时数据存储媒介；
  4. 当电源关闭时RAM不能保留数据；

返回目录

三、ROM特点

  1. ROM 称作"只读存储器"，只能读出，不能改写；
  2. ROM所存数据一般是装入整机前事先写好的；
  3. ROM断电后所存数据也不会改变。
  4. 电脑中的硬盘是可读可写的；

返回目录

四、结论

  1. RAM与ROM其实都是内存；
  2. 硬盘是外存；
  3. ROM不等于硬盘
  4. 电脑中的ROM是用来存储一些系统相关信息和开机引导BIOS;
