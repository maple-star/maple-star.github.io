linux VIM基本命令

常用的 linux基本命令

命令 功能
su - root 实现普通用户切换root用户
lsusb 查看所有的usb设备
ls -l /dev/ttyUSB* 查看当前USB设备使用的串口
xdg-open 打开各种文件方法
cd home/ 进入home目录
ls 查看当前目录下的文件
exit 退出当前状态
mkdir 新建文件夹
touch 新建文件
rmdir file 删除文件夹
rm -r 删除带有文件的文件夹
rm file 删除文件
nano file.txt 打开nano文本编辑器
ctrl+x 关闭nano文本编辑器

常用的Vim基本命令

首先你要进入Vim编辑器

命令 功能
i vim写入代码功能
esc 退出vim写入代码功能
shift+: 打开vim终端
:wq 保存代码文件
:p 退出Vim编辑器
gcc file.c -o file 生成可执行文件
./file 运行可执行文件
