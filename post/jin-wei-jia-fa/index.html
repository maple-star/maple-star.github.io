进位加法

2022-01-02

补码是用来解决负数在计算机中的表示问题的。正数的补码就是其本身；负数的补码是在其原码的基础上, 符号位不变, 其余各位取反, 最后+1. (即在反码的基础上+1)。
按位与 ：有零则零，全一出一； 清零作用，进位作用
按位异或：相同为零，不同为一；相加不进位作用

3->(b)0011
2->(b)0010

(b)0011&(b)0010 =>(b)0010<< 1 =>(b)0100
(b)0011^(b)0010 =>0001

(b)0100&(b)0001 =>(b)0000<< 1 =>(b)0000
(b)0100^(b)0001 =>(b)0101->5

int getSum(int a, int b){
 int val;
 while(a!=0)
 {
    val=(unsigned int)(a&b)<<1;//通过按位与检测进位，为防止左移后 不能在int类型中表示 强转为无符号整型
    b=a^b;//通过异或相加
    a=val;
 }
 return b;
}
